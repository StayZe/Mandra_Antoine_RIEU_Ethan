================================================================================
ANALYSE COMPARATIVE
================================================================================

Complexités temporelles:
  - Bubble Sort:    O(n²) moyen et pire cas
  - Insertion Sort: O(n²) moyen et pire cas, O(n) meilleur cas
  - Selection Sort: O(n²) dans tous les cas
  - Merge Sort:     O(n log n) dans tous les cas
  - Quick Sort:     O(n log n) moyen, O(n²) pire cas

Recommandations:
  - Pour petites listes: Insertion Sort
  - Pour grandes listes: Merge Sort ou Quick Sort
  - Pour stabilité garantie: Merge Sort
  - Pour économie de mémoire: Quick Sort (in-place)
